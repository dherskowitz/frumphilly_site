# Frum Philly
The Frum Philly weebsite source code.

## Stack
- Python 3.8
    - Black (Formatter)
    - Flake8 (Linter)
- Django 3.0.2
    - Python Decouple
    - Django All Auth
- Docker/Compose

## Get Started
- Clone repo locally
- run `pip install -r requirements.txt`
- run `docker-compose up -d --build`
- go to `192.168.99.100:8001`
